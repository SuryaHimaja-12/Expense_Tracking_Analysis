#  Smart Expense Tracker

A data-driven personal finance project that helps track and analyze income and expenses. This project uses real-world financial transaction data to uncover spending habits, identify savings opportunities, and visualize patterns through Python-based Exploratory Data Analysis (EDA) and an interactive Power BI dashboard.

---

##  Project Objectives

- Analyze personal financial data to understand spending behavior
- Identify top spending categories and monthly trends
- Monitor income vs. expenses and track savings over time
- Build a Power BI dashboard for interactive data exploration

---

##  Tools & Technologies Used

- **MySQL** – Data storage and queries  
- **Python** – Data cleaning & EDA (pandas, matplotlib, seaborn)  
- **Power BI** – Dashboard & data visualization  
- **Excel** – Initial cleaning and formatting  
- **GitHub** – Version control and project sharing

---

##  Features & Insights

- Monthly & yearly trend analysis
- Income vs. expense comparison
- Cumulative savings tracking
- Top categories of spending visualized
- Interactive filters and charts in Power BI
